Project Overview:
This project demonstrates the use of Bootstrap's grid system to create a responsive webpage layout. The webpage is built using HTML and Bootstrap to showcase a structured design that adapts to different screen sizes.

Live Demo:
Click here to view the project.

Technologies Used:
1,HTML5.
2,Bootstrap 5.
3,CSS (Bootstrap default styles).

How to Use:
1,Open the live demo link to view the project.
2,Resize your browser window to see the responsive behavior.
3,Use the browser's developer tools (F12) to inspect the Bootstrap grid classes.

Installation & Usage:
1,If you want to run this project locally:
2,Clone the repository or download the project files.
3,Open Index.html in any modern web browser.
